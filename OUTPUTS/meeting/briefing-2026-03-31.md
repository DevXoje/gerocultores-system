## Las 3 Decisiones que Necesitamos

### Decisión 1 — ⚡ URGENTE: Hosting (desbloquea Sprint 1)

**¿Firebase Hosting o GCP (Cloud Run / Cloud Functions)?**

- Firebase Hosting: más simple, integrado con Auth+Firestore, plan Spark gratuito hasta ciertos límites.
- GCP Cloud Run: más flexible, permite Express desplegado como contenedor, plan siempre-free con límites.

**Impacto**: Sin esta decisión, las tareas de despliegue de Sprint 1 (T-88..T-92) están bloqueadas.  
**Fecha límite para decidir**: Antes del final de Sprint 0 (recomendado: esta misma reunión).  
**ADR asociado**: `DECISIONS/ADR-04b-deployment-rgpd.md`
